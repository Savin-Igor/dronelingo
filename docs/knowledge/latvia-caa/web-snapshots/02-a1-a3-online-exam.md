# A1/A3 TIEŠSAISTES APMĀCĪBU KURSS UN EKSĀMENS

| Field | Value |
|-------|-------|
| Valoda | Latviešu |
| Cena | BEZMAKSAS |
| Priekšnosacījumi | Latvijā piešķirts tālvadības pilota numurs (LVA-RP-############) |
| Reģistrācija | https://e.caa.gov.lv/ |

## APRAKSTS

Persona, kura vada bezpilota gaisa kuģi, ir tālvadības pilots. Atbilstoši ES vienotā regulējuma prasībām, ja lidojumus veic ar bezpilota gaisa kuģi, kam ir **C1, C2, C3 vai C4 klases marķējums**, vai tā **masa ir vismaz 250g**, tālvadības pilotam pirms lidojumu uzsākšanas ir:

### 1. Jāapgūst attālinātais apmācību kurss

Šajā kursā iekļauta informācija, kas nepieciešama drošai bezpilota gaisa kuģu pilotēšanai atvērtās kategorijas ietvaros:
- regulējuma prasības
- aviācijas drošums
- gaisa telpas ierobežojumi
- bezpilota gaisa kuģa sistēma un tās darbība
- cilvēka veiktspējas ierobežojumi
- darbības procedūras
- privātuma un personas datu aizsardzība
- apdrošināšana
- drošība

Kursa apgūšanai **nav laika ierobežojuma** un mācību procesu iespējams jebkurā brīdī pārtraukt un turpināt no vietas, kur tas tika pārtraukts.

### 2. Sekmīgi jānokārto A1/A3 apakškategorijas attālinātais eksāmens

| Parametrs | Vērtība |
|-----------|---------|
| Ilgums | **40 minūtes** |
| Jautājumu skaits | **40 jautājumi** ar atbilžu variantiem |
| Sekmes slieksnis | Vismaz **75 % pareizu atbilžu** |

Tēmas:
- aviācijas drošums
- gaisa telpas ierobežojumi
- aviācijas regulējums
- cilvēka veiktspējas ierobežojumi
- ekspluatācijas procedūras
- vispārīgas zināšanas par UAS
- privātuma un personas datu aizsardzība
- apdrošināšana
- drošība

### Sertifikāts

Pēc sekmīgi nokārtota eksāmena saņem **apliecinājumu** par nepieciešamo zināšanu apgūšanu, **lai veiktu lidojumus A1 un A3 apakškategorijās**. Apliecinājums:
- pieejams CAA Bezpilota gaisa kuģu informācijas sistēmā;
- **derīgs 5 gadus**;
- **atzīts visā Eiropas Savienībā**.
