# ATZĪTĀS STRUKTŪRAS (Recognized Organizations / Exam Centres)

Civilās aviācijas aģentūras atzītās struktūras nodrošina:
- tālvadības pilotu **teorētisko zināšanu eksāmenu pieņemšanu** (A2, STS);
- specifiskajai kategorijai noteikto **praktisko prasmju apmācības un novērtēšanu**.

> Lai kļūtu par CAA atzīto struktūru: `caa@caa.gov.lv`.

Regulējums atvērtajā kategorijā paredz tālvadības pilotu praktisko prasmju apgūšanu pašmācības ceļā — organizācijām, kas sniedz attiecīgo pakalpojumu, CAA neapstiprina apmācību procesus un programmas.
