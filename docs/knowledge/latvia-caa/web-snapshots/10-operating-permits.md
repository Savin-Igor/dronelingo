# SPECIFISKĀS KATEGORIJAS EKSPLUATĀCIJAS ATĻAUJAS

## PIETEIKŠANĀS

Iesniegumu un pielikumus (parakstītus ar drošu elektronisko parakstu) iesniedz: e-pakalpojumu portāla "Specifiskā kategorija" → "[Ekspluatācijas atļaujas](https://e.caa.gov.lv/specific-category/operating-permits)".

Pievieno:
- ekspluatācijas risku novērtējumu (SORA vai PDRA atsauci);
- riska mazināšanas pasākumu sarakstu;
- ekspluatācijas rokasgrāmatu (UAS ekspluatants, darbības, personāla atbildība, apmācības, UAS tehniskā informācija);
- apdrošināšanas seguma apstiprinājumu;
- pakalpojuma apmaksas apliecinājumu.

## DERĪGUMS UN MAKSAS

| Pakalpojums | Maksa |
|-------------|-------|
| Atļaujas iesnieguma izvērtējums (SAIL I/II) | **175 EUR** |
| Atļaujas iesnieguma izvērtējums (SAIL III+; tikai SORA) | **325 EUR** |
| Atļaujas iesnieguma izvērtējums ar termiņu līdz 10 dienām (SORA) | **75 EUR** |
| Grozījumi atļaujā bez izmaiņām risku mazinājumā | **25 EUR** |
| Atļaujas termiņa pagarināšana | **100 EUR** |

Atļaujas derīguma termiņš: **līdz 2 gadiem**.

## IEPRIEKŠ DEFINĒTI RISKU NOVĒRTĒJUMI (PDRA)

PDRA-S = atbilst STS, izņemot prasību UAS marķēt ar attiecīgu C klases marķējumu.
PDRA-G = vispārīgāki scenāriji.

### PDRA-S01 — VLOS, kontrolēta zemes teritorija apdzīvotā vietā

| | |
|---|---|
| Lidojuma veids | VLOS |
| Pārlidojamās teritorijas | Apdzīvotās vietās virs kontrolētas zemes teritorijas |
| Gaisa telpa | Kontrolējamā un nekontrolējamā |
| Augstums | līdz 120 m AGL |
| UAS parametri | gabarīti < 3 m, kinētiskā enerģija < 34 kJ |
| C-klase | nav nepieciešams (≈ C5) |
| Pilota teorija | STS sertifikāts |
| Pilota prakse | STS-01 akreditācija (no atzītās struktūras vai UAS ekspluatanta) |

### PDRA-S02 — BVLOS ar gaisa telpas novērotāju, ārpus apdzīvotām

| | |
|---|---|
| Lidojuma veids | BVLOS ar vizuālu gaisa telpas novērošanu |
| Pārlidojamās teritorijas | Ārpus apdzīvotām vietām virs kontrolētas zemes teritorijas |
| Gaisa telpa | Kontrolējamā un nekontrolējamā |
| Augstums | līdz 120 m AGL |
| UAS parametri | < 3 m, < 34 kJ |
| C-klase | nav nepieciešams (≈ C6) |
| Pilota teorija | STS sertifikāts |
| Pilota prakse | STS-02 akreditācija |

### PDRA-G01 — BVLOS, ārpus apdzīvotām, nekontrolējamā gaisa telpā

| | |
|---|---|
| Lidojuma veids | BVLOS ar vizuālu gaisa telpas novērošanu |
| Pārlidojamās teritorijas | Ārpus apdzīvotām vietām |
| Gaisa telpa | Nekontrolējamā |
| Augstums | līdz 150 m AGL |
| UAS parametri | < 3 m, < 34 kJ |
| C-klase | nav nepieciešams |
| Pilota | STS teorija + STS-02 prakse |

### PDRA-G02 — BVLOS rezervētā gaisa telpā

| | |
|---|---|
| Lidojuma veids | BVLOS īpaši rezervētā gaisa telpā |
| Pārlidojamās teritorijas | Ārpus apdzīvotām vietām |
| Gaisa telpa | Rezervēta |
| Augstums | atbilstoši rezervācijai |
| UAS parametri | < 3 m, < 34 kJ |
| C-klase | nav nepieciešams |
| Pilota | STS teorija + STS-02 prakse (var izlaist novērotāja un VLOS elementus) |

### PDRA-G03 — BVLOS atipiskā gaisa telpā (ARC-a)

ARC-a nosacījumi:
1. lidojumi rezervētā gaisa telpā; vai
2. < 30 m AGL ārpus lidlaukiem; vai
3. šķēršļu tiešā tuvumā (≤ 15 m vertikāli, 30 m horizontāli; mākslīgais šķērslis līdz 20 m augstumā — vertikāli līdz 30 m).

UAS: < 3 m, < 34 kJ. Pilota teorija: STS sertifikāts. Pilota prakse: STS-02 vai līdzvērtīga (var izlaist novērotāja, VLOS elementus).

UAS ekspluatants reģistrējas valstī, kur ir tā deklarētā dzīvesvieta vai uzņēmējdarbības vieta. Lai lidot ārpus reģistrācijas valsts — pieteikums attiecīgajai NAA ar reģistrācijas valstī izdotās atļaujas kopiju (Regulas (ES) 2019/947 13. pants).
