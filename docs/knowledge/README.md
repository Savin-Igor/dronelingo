# Knowledge Base — Drone Pilot Training (EU / Latvia)

Curated PDF source documents for EASA Open Category (A1/A3, A2, STS) drone
pilot certification, with a focus on Latvia and the EASA zone.

All files were downloaded on 2026-05-08. Treat originals on issuing-authority
sites as the source of truth — these are local snapshots.

## Notes on missing items

- **The official CAA Latvia question bank** is **not published**. Theory
  exams are taken interactively at `e.caa.gov.lv`. Only training material
  and sample questions (above) are available.
- **EUR-Lex direct PDFs:** `eur-lex.europa.eu` is fronted by CloudFront WAF
  that blocks scripted downloads. Wikimedia Commons hosts authentic OJ copies
  of the original 2019/947 and 2019/945 — used as substitute.
- **Latvian Cabinet Regulations (MK noteikumi):** `likumi.lv` serves HTML, not
  PDF. URLs above point to the official live versions.

## Reproducibility

To re-fetch a single file, copy the URL from the corresponding row's "Source"
column and download with curl. Most sources are public and stable; eur-lex.
europa.eu requires a real browser (use firecrawl with a stealth proxy or the
Wikimedia Commons mirror).
